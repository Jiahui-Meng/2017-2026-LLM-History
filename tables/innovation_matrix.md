# Innovation Matrix

| 创新方向 | 代表节点 | 关键说明 |
|---|---|---|
| self-attention | Transformer | 现代 LLM 统一底座。[S036] |
| generative pre-training | GPT-1 | 先预训练再迁移的主线成形。[S037] |
| bidirectional understanding | BERT | 编码器路线在理解任务上奠基。[S038] |
| unified text interface | T5 | 多任务统一为 text-to-text。[S039] |
| scaling / in-context learning | GPT-3 | 规模带来 few-shot 涌现。[S040] |
| instruction tuning | FLAN | 模型更会执行自然语言任务要求。[S042] |
| compute-optimal training | Chinchilla | 参数与 token 配比优化成为关键。[S041] |
| RLHF | InstructGPT / ChatGPT | 对齐与产品可用性显著提升。[S043] |
| Constitutional AI | Claude | 将规则化自我批评推入主流。[S045] |
| open weights | Llama 2 | 开放权重生态大规模扩展。[S049] |
| sparse MoE | Mixtral / Llama 4 / Mistral Large 3 | 开源和前沿部署都采用稀疏专家路线。[S056][S033][S025] |
| long context | Gemini 1.5 / GPT-5.4 / Claude Opus 4.7 / Kimi K2.6 | 长上下文从演示能力变成产品能力。[S048][S003][S006][S017] |
| multimodal | GPT-4 / Gemini / Llama 4 / Kimi K2.6 / Qwen3.x | 图像、视频、文档进入统一模型路线。[S044][S008][S033][S027][S030] |
| MLA / DSA | DeepSeek-V3.2 / GLM-5.1 | 关注长上下文和推理效率的结构优化。[S028][S032] |
| reasoning RL | DeepSeek-R1 | 把推理过程训练做成显式路线。[S057] |
| agent / tool use | Claude 4.x / Gemini 3 / GLM-5.1 / MiniMax-M2.1 | 工具调用与 long-horizon work 成为核心能力。[S005][S009][S020][S022] |
