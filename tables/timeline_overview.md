# LLM Timeline Overview

| 时间 | 节点 | 机构/家族 | 比上一节点提升 | 核心创新 | 行业影响 |
|---|---|---|---|---|---|
| 2017-06 | Transformer | Google Research | 训练并行性更强，长程依赖处理更好 | self-attention | 现代 LLM 共同底座。[S036] |
| 2018-06 | GPT-1 | OpenAI | 通用迁移范式更清晰 | generative pre-training | GPT 系列起点。[S037] |
| 2018-10 | BERT | Google | 理解任务表现大幅提升 | bidirectional MLM | 编码器路线确立。[S038] |
| 2019-10 | T5 | Google | 多任务统一接口 | text-to-text | 影响 instruction tuning。[S039] |
| 2020-05 | GPT-3 | OpenAI | few-shot 能力增强 | scaling + in-context learning | 规模涌现共识形成。[S040] |
| 2021-10 | FLAN | Google | 更会按指令工作 | instruction tuning | 后训练成为一等能力。[S042] |
| 2022-03 | Chinchilla | DeepMind | 同算力训练更优 | compute-optimal scaling | 改变训练经济学。[S041] |
| 2022-12 | ChatGPT / InstructGPT | OpenAI | 对话、可用性和遵循意图大幅增强 | RLHF 产品化 | LLM 进入大众市场。[S043] |
| 2023-03 | Claude / Constitutional AI | Anthropic | 安全与可控性强调更强 | Constitutional AI | 对齐路线分化。[S045][S046] |
| 2023-03 | GPT-4 | OpenAI | 泛化和多模态提升 | frontier closed model system design | 闭源前沿门槛提升。[S044] |
| 2023-07 | Llama 2 | Meta | 开放权重生态加速 | open-weight family | 开源生态爆发。[S049] |
| 2023-12 | Mixtral 8x7B | Mistral | 更高总容量、更低激活成本 | sparse MoE | 开源 MoE 主流化。[S056] |
| 2024-02 | Gemini 1.5 | Google | 长上下文和多模态大幅前进 | 1M context | 长上下文产品化。[S048] |
| 2024 | Qwen2 / Qwen2.5 | Qwen | 中文/多语言/代码均衡提升 | fast-iterating open family | Qwen 成为核心开源家族。[S058][S059] |
| 2024-12 | DeepSeek-V3 | DeepSeek | 能力/效率/透明度更强 | DeepSeekMoE + MLA + MTP | 开源 frontier 结构创新出圈。[S015] |
| 2025-01 | DeepSeek-R1 | DeepSeek | 推理能力和过程优化增强 | reasoning-first RL | 行业重估 reasoning RL。[S057] |
| 2025-2026 | Claude 4.x | Anthropic | long-horizon coding/agent 更强 | effort + task budget | 推理预算显式化。[S005][S047] |
| 2025-2026 | Qwen3.x | Qwen | MoE、多模态、部署与能力升级 | hybrid attention + MoE | 开源家族继续前沿化。[S019][S060] |
| 2025-2026 | Kimi K2.x | Kimi | 长文本进化为 agent 工作流 | multimodal + thinking modes | Kimi 路线升级为综合 agent。[S017][S018][S061] |
| 2025-2026 | Llama 4 / Muse Spark | Meta | 基础模型与产品系统双线推进 | MoE multimodal + product routing | Meta 路线分叉清晰化。[S011][S033][S034] |
| 2025-2026 | Gemini 3 / GPT-5.x / GLM-5.1 / MiniMax-M2.1 / Grok 4.20 / Mistral Large 3 | 多家 | 工具、推理、长上下文、部署集体升级 | reasoning + agent + sparse deployment | 当前竞争转向 long-horizon work。[S001][S009][S020][S022][S024][S025] |
