# Agentic Tool Use

Agentic 能力不能只看模型权重。GPT-5.4 的 computer use、tool search、web research 是 API/Codex/ChatGPT 系统能力。[S003: Computer use and vision] Claude Opus 4.7 的 Claude Code、task budgets、auto mode 与 ultrareview 也是产品与模型共同构成。[S005: Also launching today] Gemini 3 的 function calling、code execution、file search、search grounding 是 Gemini API capability。[S009: Model table] GLM-5.1 的 MCP/function/structured/context cache 是 Z.ai 平台能力。[S022: Capability]
