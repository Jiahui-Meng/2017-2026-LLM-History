# Grok 家族发展史

## 家族主线

Grok 的公开材料比开源家族少得多，因此更适合作为“产品化闭源前沿路线”的观察对象。

| 时间 | 节点 | 提升 | 创新 |
|---|---|---|---|
| 2026 | Grok 4.20 | 长上下文与 reasoning/non-reasoning 分层更明显 | 2M context + route variants。[S024] |

## 当前最新节点

- `Grok 4.20` 官方公开重点是 2M context、reasoning/non-reasoning 变体和 API 产品接口，而不是内部结构。[S024]

## 仍未披露的信息

参数、层数、专家数、训练配方、对齐细节官方未披露。[S024]
