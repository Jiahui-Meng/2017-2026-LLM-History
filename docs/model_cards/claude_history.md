# Claude 家族发展史

## 家族主线

Claude 的独特之处不只是“更强聊天”，而是把 Constitutional AI、长上下文和 agentic coding 逐步做成系列主标签。

| 时间 | 节点 | 提升 | 创新 |
|---|---|---|---|
| 2023 | Claude 1 | 更强调安全、可控与宪法式对齐 | Constitutional AI 主流化。[S045] |
| 2023 | Claude 2 | 长上下文与企业可用性增强 | 更强文档处理与长文本交互。[S046] |
| 2024 | Claude 3 | 家族分层更清晰，能力上限提升 | Haiku / Sonnet / Opus 产品线成型。[S047] |
| 2025-2026 | Claude 4 / Opus 4.7 | 长程执行、agentic coding、effort 控制更强 | xhigh effort、task budget、agent workflow。[S005][S006] |

## 每代的核心创新

- `Claude 1`：把 Constitutional AI 做成与 RLHF 并列的公开对齐路线。[S045]
- `Claude 2`：把“安全而长文本可用”推进到更成熟的产品层。[S046]
- `Claude 3`：让 Claude 家族从单模型叙事升级为分层产品线。[S047]
- `Claude 4.x`：把推理预算和 agentic coding 显式化，开始正面竞争 long-horizon work。[S005]

## 当前最新节点

- `Claude Opus 4.7` 是当前公开主线中的高端节点，官方强调 1M context、xhigh effort、更高分辨率图像输入和 agentic coding 改进。[S005][S006]

## 仍未披露的信息
