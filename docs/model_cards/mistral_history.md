# Mistral 家族发展史

## 家族主线

Mistral 的历史重点是“欧洲开源/开放权重路线 + 稀疏 MoE + 企业部署”。

| 时间 | 节点 | 提升 | 创新 |
|---|---|---|---|
| 2023 | Mixtral 8x7B | 开源侧以较低激活成本获得更高容量 | 稀疏 MoE 进入主流开源。[S056] |
| 2025-2026 | Mistral Large 3 | 多模态、675B/41B active、企业部署能力增强 | sparse MoE frontier 化。[S025][S026][S035] |

## 当前最新节点

- `Mistral Large 3` 是当前公开主线旗舰，官方强调 Apache 2.0、675B total / 41B active、256k context 和多模态能力。[S025][S026]
